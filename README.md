# Customer Churn Predictor

A Machine Learning web application built with Streamlit that predicts whether a customer will churn or stay, along with a confidence percentage. Built using Random Forest algorithm from Scikit-learn.

---

## Project Overview

This project takes customer details as input — tenure, monthly charges, number of complaints, and plan type — and predicts churn using a trained Random Forest model. No API key is required. Everything runs locally on your machine.

---

## Project Structure

```
Customer_Churn_Predictor/
│
├── generate_data.py          # Generates synthetic customer dataset
├── train_model.py            # Trains the Random Forest model
├── app.py                    # Streamlit web application
├── churn_data.csv            # Auto-generated dataset (created after running generate_data.py)
└── churn_predictor_model.pkl # Trained model file (created after running train_model.py)
```

---

## Required Packages

Install the following packages using pip in your terminal:

```
pip install pandas
pip install numpy
pip install scikit-learn
pip install streamlit
```

Note: The `pickle` library does not need to be installed. It is built into Python.

---

## How to Run the Project

Follow these steps in order. Do not skip steps.

**Step 1 — Generate the dataset**

Run this file first. It creates `churn_data.csv` in your project folder.

```
python generate_data.py
```

Expected output:
```
Dataset created with 1000 rows
churn
0    613
1    387
dtype: int64
```

**Step 2 — Train the model**

Run this file after Step 1. It reads the CSV, trains the model, and saves `churn_predictor_model.pkl`.

```
python train_model.py
```

Expected output:
```
Model Accuracy: 83.xx%
Model saved as churn_predictor_model.pkl
```

**Step 3 — Launch the web app**

Run this using the `streamlit run` command. Do not use the PyCharm run button for this step.

```
streamlit run app.py
```

Your browser will open automatically at:
```
http://localhost:8501
```

---

## How the App Works

1. Open the app in your browser
2. Use the sliders to enter customer details
3. Click the Predict button
4. The app displays either CHURN or STAY with a confidence percentage

---

## Input Fields

| Field | Range | Description |
|-------|-------|-------------|
| Tenure | 1 to 72 months | How long the customer has been with the company |
| Monthly Charges | 20 to 120 | Amount the customer pays per month |
| Number of Complaints | 0 to 7 | Total complaints raised by the customer |
| Plan Type | Basic, Standard, Premium | The customer's current subscription plan |

## Important Notes

- Always run `generate_data.py` before `train_model.py`
- Always run `train_model.py` before `app.py`
- Use `streamlit run app.py` in the terminal — not the PyCharm run button
- The model accuracy may vary slightly between runs but should stay between 80% and 88%
- This project uses synthetic data generated by `generate_data.py`, not real customer data
